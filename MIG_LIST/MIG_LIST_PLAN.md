# MIG_LIST - Excel Upload to MIG_TAB_LIST

## Overview
Excel 파일을 업로드하면 DBADM.MIG_TAB_LIST INSERT SQL 파일을 생성하는 Streamlit 웹 페이지

## Tech Stack
- **Frontend/Backend**: Python Streamlit
- **Excel Parsing**: openpyxl (xlsx), pandas
- **Output**: INSERT SQL 파일 생성 (DB 직접 접속 없음)

## Excel Template Format

| Column | Description | Required |
|---|---|---|
| OWNER | Schema name | Y |
| TABLE_NAME | Table name | Y |
| WHERE_COL1 | 1st condition column | N |
| PRE1 | 1st condition value (>= comparison) | N |
| WHERE_COL2 | 2nd condition column | N |
| PRE2 | 2nd condition value (< comparison) | N |

## Features

### 1. Excel Upload
- xlsx 파일 드래그 앤 드롭 또는 파일 선택
- 업로드 즉시 데이터 미리보기 테이블 표시

### 2. Data Validation
- OWNER, TABLE_NAME 필수값 체크
- WHERE_COL1 없이 PRE1만 있는 경우 경고
- WHERE_COL2 없이 PRE2만 있는 경우 경고
- 중복 OWNER+TABLE_NAME 체크

### 3. Preview
- 파싱된 데이터를 테이블로 표시
- WHERE 조건 조립 결과 미리보기
- 유효성 검사 결과 (Error/Warning) 표시

### 4. SQL Generation
- TRUNCATE + INSERT 구문 생성
- NULL 처리 (빈 셀 -> NULL)
- 다운로드 버튼으로 .sql 파일 저장

## Generated SQL Example

```sql
-- Generated by MIG_LIST
-- Date: 2026-05-14 14:30:00
-- Total: 3 rows

TRUNCATE TABLE DBADM.MIG_TAB_LIST;

INSERT INTO DBADM.MIG_TAB_LIST (OWNER, TABLE_NAME, WHERE_COL1, PRE1, WHERE_COL2, PRE2)
VALUES ('DBADM', 'TEST_ORDER', 'ORDER_DATE', '20250401', NULL, NULL);

INSERT INTO DBADM.MIG_TAB_LIST (OWNER, TABLE_NAME, WHERE_COL1, PRE1, WHERE_COL2, PRE2)
VALUES ('DBADM', 'TEST_LOG', 'LOG_DATE', '20250201', 'LOG_DATE', '20250601');

INSERT INTO DBADM.MIG_TAB_LIST (OWNER, TABLE_NAME, WHERE_COL1, PRE1, WHERE_COL2, PRE2)
VALUES ('DBADM', 'TEST_CODE', NULL, NULL, NULL, NULL);

COMMIT;
```
